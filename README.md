# My Nix Config

## Features:
 - Minimal Sway environment
 - Basic neovim setup
 - Sound, Network/Wifi, Bluetooth
 - Requires minimal effort to install/customize
 - Installs Steam & protontricks-launch fix out of the box

## Caveats
 - Customized for MY usecase
 - Need to remove/change appropiate sections like mounting drives
 - May feel bloated for purists
 - Does not use flakes or homemanager, less "optimized" or "declarative"
